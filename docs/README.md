# Part 2: 도메인 데이터 분석

AI 서비스에 필요한 데이터 분석 방법을 학습합니다. 코드를 예시로 사용하지만, 같은 원리가 문서, 로그, 설정 파일 등 모든 구조화된 데이터에 적용됩니다.

---

## 학습 목차

| STEP | 제목 | 핵심 개념 |
|------|------|----------|
| [07](STEP-07.md) | 파싱 | AST, JavaParser |
| [08](STEP-08.md) | 구조 분석 | findAll, 메트릭 |
| [09](STEP-09.md) | 의미 분석 | 심볼 테이블, 스코프 |
| [10](STEP-10.md) | 패턴 매칭 | 코드 스멜, 안티패턴 |
| [11](STEP-11.md) | 이슈 탐지 | 보안, 버그, OWASP |
| [12](STEP-12.md) | 점수화 | 품질 점수, 등급 |

---

## 전체 흐름

```
파싱 → 구조화 → 의미분석 → 패턴 → 이슈 → 점수
```

```
[데이터 입력 (코드)]
     ↓
[STEP 7] 파싱: 텍스트 → 트리 구조
     ↓
[STEP 8] 구조 분석: 메트릭 추출
     ↓
[STEP 9] 의미 분석: 관계 추적
     ↓
[STEP 10] 패턴 매칭: 문제 패턴 탐지
     ↓
[STEP 11] 이슈 탐지: 위험 요소 발견
     ↓
[STEP 12] 점수화: 정량화된 결과
     ↓
[분석 결과 → AI에게 전달]
```

---

## 핵심 키워드

`파싱` `AST` `메트릭` `패턴 매칭` `규칙 기반 분석` `점수화`

---

## 학습 후 할 수 있는 것

- 텍스트를 구조화된 형태로 변환
- 데이터에서 필요한 정보 추출
- 규칙 기반으로 패턴 자동 탐지
- 분석 결과를 AI에게 전달할 형태로 정량화
