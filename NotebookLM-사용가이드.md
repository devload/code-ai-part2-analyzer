## 2. NotebookLM 사용 방법

### Step 1: 노트북 생성
1. https://notebooklm.google.com 접속
2. "New notebook" 클릭
3. 제목: "토큰으로 이해하는 AI - 발표 자료"

### Step 2: 소스 추가
1. "Add source" 클릭
2. `NotebookLM-소스-토큰과AI.md` 파일 업로드
3. (선택) 보조 문서들도 추가

### Step 3: PPT 생성 프롬프트

NotebookLM에 다음과 같이 요청하세요:

```
이 문서를 바탕으로 30-40분 발표용 PowerPoint 자료를 만들어주세요.

발표 구조:
1. Opening (5분)
   - 우리가 겪는 AI의 신비로운 현상들
   - 발표 목표

2. Part 1: 토큰이란? (5분)
   - 토큰 정의
   - 실제 토큰화 예제
   - 비용과의 관계

3. Part 2: 다음 토큰 예측 (7분)
   - AI의 핵심 동작
   - Bigram 소개
   - 학습과 생성 과정

4. Part 3: 실전 프로젝트 (5분)
   - Mini AI 구조
   - Step별 여정
   - 핵심 코드

5. Part 4: 토큰으로 설명되는 현상들 (8분)
   - 비용
   - 속도
   - 기억
   - 다양성

6. Part 5: Bigram → Trigram (5분)
   - Bigram의 한계
   - Trigram의 필요성
   - 희소성 문제

7. Part 6: Bigram → GPT (3분)
   - 비교표
   - 공통점과 차이점

8. Part 7: 실습 체험 (2분)
   - 빠른 시작 스크립트
   - 교육용 데이터셋

9. Closing (2분)
   - 핵심 Takeaways
   - 다음 단계

슬라이드 디자인 요청:
- 각 Part는 섹션 타이틀 슬라이드로 구분
- 코드 예제는 syntax highlighting
- 비교표는 표 형식으로
- 핵심 인사이트는 강조 박스로
- 다이어그램은 텍스트 기반으로 시각화
```

## 4. 슬라이드 디자인 팁

### 시각적 요소

**다이어그램 제안**:
1. 토큰화 과정
   ```
   "오늘은 날씨가 좋다"
        ↓ 토큰화
   ["오늘은", "날씨가", "좋다"]
        ↓ 숫자 변환
   [101, 205, 309]
   ```

2. 다음 토큰 예측 루프
   ```
   Start → 토큰1 → 토큰2 → ... → 토큰N → End
      ↑                                   ↓
      └──────── 반복 ────────────────────┘
   ```

3. Bigram 카운트 테이블
   ```
   오늘은 → 날씨가: ■■■ (3회)
   오늘은 → 기분이: ■■■ (3회)
   날씨가 → 좋다:   ■■■■■ (5회)
   ```

4. 문맥 길이 비교
   ```
   Bigram:   [1개] → 다음
   Trigram:  [2개] → 다음
   GPT-4:    [수만개] → 다음
   ```

### 색상 코딩
- 🟦 파란색: 토큰, 입력
- 🟩 초록색: 학습, 성공
- 🟨 노란색: 주의, 인사이트
- 🟥 빨간색: 문제, 한계

### 강조 포인트
- 💡 핵심 인사이트
- ✅ 완료/성공
- ❌ 문제/한계
- 🎯 목표/포커스

## 5. 발표 시 데모 계획

### 라이브 데모 (선택)

**데모 1: 토큰화** (1분)
```bash
./gradlew :mini-ai-cli:run --args="tokenize '할 수 있다'"
./gradlew :mini-ai-cli:run --args="tokenize '할수있다'"
```

**데모 2: 빠른 체험** (2분)
```bash
./examples/빠른시작-토큰체험.sh
```
- 실시간으로 7가지 개념 시연
- 청중이 직접 보면서 이해

**데모 3: 문장 생성** (1분)
```bash
./gradlew :mini-ai-cli:run --args="run -p '오늘은' --max-tokens 10 --seed 42"
```
- 같은 seed로 재현성 보여주기
- 다른 seed로 다양성 보여주기

### 데모 없이 진행
- 슬라이드에 스크린샷 포함
- 결과 미리 캡처해서 보여주기
